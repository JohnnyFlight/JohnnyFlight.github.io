Learning Game

This is just a simple game I made with the intention of using it to input into a reinforcement learning neural network

SCORE:

Rules

  • Enemies spawn every 5 turns
  • Powerups spawn every 7 turns
  • Powerups last for 2 turns (not including the turn you get it)
  • Collecting multiple powerups does not stack powerup time
  • Enemies move every 3 turns
  • Enemies destroy powerups they move through
  • Touch enemies with powerup to kill them
  • Touch enemies without powerup to die
